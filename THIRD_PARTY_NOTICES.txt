ClearBridge Modifications
-------------------------

The ClearBridge team added or modified substantial functionality, including:

- Structured crisis-to-action analysis.
- Multilingual English, Simplified Chinese, and Arabic UI/output support.
- OCR review workflows and OCR quick action card.
- Manual caption range analysis.
- Rolling Summary overlay and memory-only temporary context.
- Human-in-the-loop confirmation and local History categories.
- Source-evidence validation, test harnesses, and competition documentation.

Windows OCR and Live Captions
-----------------------------

ClearBridge uses Windows platform capabilities such as Windows.Media.Ocr and
Windows Live Captions where available. These platform features are provided by
Windows and are not redistributed as ClearBridge source code.

AI and External Services
------------------------

ClearBridge can optionally use user-configured OpenAI-compatible providers,
translation providers, and cloud OCR providers at runtime. Release packages do
not include user API keys, authorization headers, setting.json, user databases,
logs, or raw user data.

Users are responsible for reviewing provider terms and privacy policies before
sending sensitive content to an online service.
